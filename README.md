## Running From Source
1. Clone this repository
2. Clone the [SakuyaEngine](https://github.com/novialriptide/SakuyaEngine) repository
3. Move the [SakuyaEngine](https://github.com/novialriptide/SakuyaEngine) directory to `~/Helix`
4. Run `main.py`

## Dependencies
```
pygame==2.1.0
```
